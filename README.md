# VOO Citizen App

A mobile and web application for citizens to report community issues with photo uploads.

## Tech Stack
- **Mobile**: Flutter (iOS/Android)
- **Web**: Next.js 14
- **Backend**: Node.js/Express
- **Database**: MongoDB (shared with VOO USSD)
- **Images**: ImgBB (free)
- **Notifications**: Supabase

## Getting Started

### Mobile App (Flutter)
```bash
cd mobile
flutter pub get
flutter run
```

### Web App (Next.js)
```bash
cd web
npm install
npm run dev
```

### Backend API
```bash
cd api
npm install
npm run dev
```
