# VOO Citizen App

The official mobile application for VOO Ward citizens.

## Latest Release (v12.0.0)

**Download the latest version here:**
[Download VOO Citizen App v12.0.0 (APK)](https://github.com/MuthamiM/voo-citizen-app/releases/latest)

### Features

- Report and track community issues
- Bursary applications
- Lost ID reporting
- Emergency contacts
- Push notifications
- Google Sign-In support

### Installation

1. Download the APK file from the releases page.
2. Allow installation from unknown sources if prompted.
3. Install and login.
